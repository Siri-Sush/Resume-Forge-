## рЯЪА How to Deploy on GitHub Pages (Step-by-Step)

### What you need
- A free [GitHub account](https://github.com)
- These 3 files: `index.html`, `style.css`, `app.js`

---

### Step 1 вАФ Create a new GitHub repository

1. Go to [github.com](https://github.com) and sign in
2. Click the **"+"** icon in the top right вЖТ **"New repository"**
3. Name it: `resume-forge` (or anything you like)
4. Make sure it is set to **Public**
5. Leave everything else as default
6. Click **"Create repository"**

---

### Step 2 вАФ Upload your files

1. On your new repository page, click **"uploading an existing file"** (the link in the middle of the page)
2. Drag and drop all 3 files into the upload area:
   - `index.html`
   - `style.css`
   - `app.js`
3. Scroll down to **"Commit changes"**
4. Leave the message as-is and click **"Commit changes"**

---

### Step 3 вАФ Enable GitHub Pages

1. On your repository page, click **"Settings"** (top menu)
2. In the left sidebar, scroll down and click **"Pages"**
3. Under **"Source"**, click the dropdown and select **"Deploy from a branch"**
4. Under **"Branch"**, select **"main"** and keep the folder as **"/ (root)"**
5. Click **"Save"**

---

### Step 4 вАФ Get your live URL

1. Wait about **60 seconds** for GitHub to build the site
2. Refresh the Settings вЖТ Pages page
3. You will see a green box that says:

   > **Your site is live at: `https://YOUR-USERNAME.github.io/resume-forge/`**

4. Click the link вАФ your app is live! рЯОЙ

Share that link with anyone вАФ it works on desktop and mobile.
